# G3nosss Portfolio

A professional, storytelling-style developer portfolio built with Next.js 14, TypeScript, Tailwind CSS, and Framer Motion.

## 🚀 Setup

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## 🛠 Tech Stack

- **Next.js 14** — App Router, server components
- **TypeScript** — Full type safety
- **Tailwind CSS** — Utility-first styling with custom theme
- **Framer Motion** — Scroll-triggered and entry animations
- **React Icons** — Consistent icon set

## ✨ Features

- 📖 **Storytelling layout** — Chapter-style sections that flow naturally
- 🎨 **Sophisticated dark theme** — Warm gold + muted steel blue accents
- 💫 **Framer Motion animations** — Scroll-triggered fade-ups, staggered children, typewriter effect
- 📱 **Fully responsive** — Mobile-first, works on all screen sizes
- 🔢 **Glassmorphism navbar** — Fixed with active section detection
- ⌨️ **Typewriter hero** — Animated code block in the hero section
- 🧩 **Modular components** — Clean, reusable architecture
